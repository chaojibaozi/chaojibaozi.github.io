# 搜索词分析系统测试报告

## 发现的严重问题

### 问题1：品牌词提取逻辑严重错误
系统将"日本"、"留学"、"中介"等普通词语误提取为品牌词。
- BRAND_WORDS = [jp, 恒企, 日本, 本留, 留学, 学中, 中介]
- 其中"日本"、"本留"、"留学"、"学中"、"中介"是n-gram碎片，根本不是品牌名
- **根因**：`extractBrandAndProductWords`中的group-based ngram分析，从包含"品牌"字的推广组关键词中提取了所有子串加入品牌词

### 问题2：产品词提取含无意义碎片
PRODUCT_WORDS包含"计培"、"会计培"等无意义的n-gram碎片

### 问题3：品牌分支打分过高（base=8+2=10）
一旦搜索词包含被误判为品牌词的普通词语（如"留学"），直接给10分

### 问题4：产品词惩罚机制过于激进（-3）
搜索词不包含产品词时扣3分，导致相关但用词不同的搜索被低估

### 问题5：无真正的语义理解
无法区分"日本留学中介" vs "美国留学中介"（都是10分）

## 结论
系统存在**严重缺陷**，需要优化语义匹配逻辑。主要优化方向：
1. 修复品牌词提取逻辑（去除group-based ngram提取）
2. 降低品牌分加成
3. 改善产品词惩罚机制
4. 增加真正的语义相似度判断
